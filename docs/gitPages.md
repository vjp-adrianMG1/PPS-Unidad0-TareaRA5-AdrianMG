# GitHub Pages

## Objetivo
Publicar la documentación generada con MkDocs en una página web accesible desde GitHub Pages.

## Configuración

1. Se activó GitHub Pages desde la pestaña **Settings > Pages** del repositorio.
2. Se seleccionó la rama `gh-pages` como fuente, con la opción `(root)` como carpeta.
3. El workflow de GitHub Actions se encargó de generar y subir los archivos estáticos a esa rama.
